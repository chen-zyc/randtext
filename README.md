# randtext
生成随机文本的一个 Rust 程序。

## 使用

### 帮助文档

```sh
> ./randtext -h

USAGE:
    randtext [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -n, --nums <num-of-chars>     [default: 10]
```

### 生成 n 个中文

```sh
> ./randtext -n 20

伃齹蟦訂枔威瀔珤婘寭闃臑穵爑蓓髐叢回秹荍
```
